- 函数表达：对于问题T，先尝试写出函数公式 T(n)；表达的时候考虑的是最坏的情况。
- 化简：化简常数项，只保留最高阶
  - 比如T(n)=3n, 则对应的时间复杂度O(n)=n
       比如T(n)=0.5*n^2+0.5n, 则对应的时间复杂度O(n)=n^2
 
- 复杂度排序
$O(1) < O(logN) < O(N) < O (N*logN ) < O (N^2 ) < O (N^3 ) < O ( 2*N ) < O (3*N )（K为常数）$
